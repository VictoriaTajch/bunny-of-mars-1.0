Comisiones

Envié su propuesta mediante esta página web en la sección ''Contacto'', una vez que se haya hecho y aceptado el presupuesto y la idea se deberá de abonar primero 35% del pago, se procederá a hacer el sketch de la idea, una vez terminado el sketch se le mostrará y se podrá discutir de arreglar algunas cosas y/o agregar algo más. Una vez obtenido el ok de esta primera fase para continuar a la siguiente se deberá de abonar el resto del pago.

Dispuesto a hacer

  • Fondos complejos
  • Semi-desnudos
  • Diseño de tatuajes
  • Sfw
  • Animales
  • Varios personajes interactuando
  • Ilustraciones para cuentos
